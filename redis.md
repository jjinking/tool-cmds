# Intro

Remote dictionary server

Default port: 6379

# Install

```bash
# Mac
brew install redis
```

# Running

```bash
# Start redis
redis-server
```

Creates a file dump.rdb

# CLI

```bash
# Start
redis-cli
redis-cli -h <host> -p <port>

# Check connection
ping # returns PONG

# Quit CLI
quit

```
